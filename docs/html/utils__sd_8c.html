Initialise un t_tab_mat_str_dyn avec un CSV.

Alloue la place nécessaire dans un t_tab_mat_str_dyn d'après le contenu d'un CSV et le remplis avec ce dernier. La ligne 0 du tableau sera la ligne d'en-tête du CSV.

Paramètres

[out]tabmots: Le t_tab_mat_str_dyn
[in]separator: Le séparateur entre les colonnes du CSV
